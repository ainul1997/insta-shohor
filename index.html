How dose javascript work?

Javascript হলো একটি ক্লায়েন্টসাইড স্ক্রিপ্টিং ভাষা। যা দক্ষতার দিক দিয়ে সকল ব্যবহৃত স্ক্রিপ্টিং ভাষাগুলো মধ্যে অন্যতম। Javascript কাজ করে ওয়েব ব্রাউজারে ওয়েব পেজে HTML পার্স করে DOM এর বিষয়বস্তু লোড করে লাইভ উপস্থাপন করে। ব্রাউজারটির সাথে লিঙ্ক করা Html এবং css গুলো পার্সার থেকে আসে। Html এবং css কে DOM দ্বারা একত্রিত করা হয় ওয়েব পেজ তৈরি করার জন্য। তখন ব্রাউজারের ইঞ্জিন javascript ফাইল ইনলাইন কোড লোড করে না চালিয়ে এটি html,css ফাইল লোডিং হওয়ার জন্য অপেক্ষা করে। একবার লোড হয়ে গেলে কোডটি কার্যকর করা হয়। Dom জাভাস্ক্রিপ্ট কোড দিয়ে আপডেট ও ব্রাউজারে রেন্ডার করা হয়।

What does javascript do?

javascript হলো একক - থ্রেডেড পোগ্রামিং ভাষা। জাভাস্ক্রিপ্ট একটি সময়ে একাধিক কাজ করতে পারেনা। প্রধান থ্রেড যেখানে জাভাস্ক্রিপ্ট কোড চালানো হয়, এক সময়ে একাধিক লাইনে বা সমান্তরালভাবে কোড চালানো সম্ভাবনা নেই। জাভাস্ক্রিপ্ট ইঞ্জিন ফাইলের উপরে থেকে নিচের দিকে কাজ করে। ফাংশন কার্যকর করতে দীর্ঘ সময় নেয়, ফাংশনটি কার্যকর করা সময় ওয়েব ব্রাউজারের সাথে যোগাযোগ করতে না পারলে পৃষ্ঠাটি হ্যাং করে। ফাংশনটি সম্পূর্ণ হতে অনেক সময় লাগে তাকে ব্লকিং ফাংশন বলে। টেকনিক্যালি, একটি ব্লকিং ফাংশন ওয়েবপৃষ্ঠার সমস্ত মিথস্ক্রিয়াকে ব্লক করে, যেমন মাউস ক্লিক।
